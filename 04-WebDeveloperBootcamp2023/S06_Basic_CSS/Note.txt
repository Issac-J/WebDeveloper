
[ Q. What is CSS ? ] 
CSS is a language for describing how documents are presented visually

* HTML : Nouns
* CSS : Adjectives
* Javascript : Verb

[ CSS Rules ]
* Example
    h1 {
        color: purple
    }

[ Style을 표현하기 위한 방법 ]
1. HTML 요소에 직접 스타일을 작성: Inline Styles
-> 간단하게 사용하는데는 괜찮지만, 좋은 방법은 아니라고 한다.

    Example
    <h1 style="color: purple">Hello World</h1>
    <button style="background-color: palegreen">BTN</button>

2. Style Element

3. External StyleSheet ( 외부 참조 스타일 )
    head에 link Element 를 사용하여 외부의 스타일을 참조한다.
